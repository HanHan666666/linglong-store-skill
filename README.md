# 玲珑应用商店 Skill

这个 Skill 用来让 AI 助手帮你完成玲珑应用商店常用操作：

- 搜索应用
- 查看应用详情
- 安装应用
- 卸载应用
- 检查更新
- 升级应用
- 反馈问题

## 安装这个 Skill

把本仓库里的 skill 目录复制到你的 Agent 技能目录：

```bash
npx skills add HanHan666666/linglong-store-skill
```

复制完成后，重启或重新加载你的 Agent 会话即可。

## 怎么和 AI 对话使用

直接用自然语言说需求即可，下面是常见示例。

### 1) 搜索应用

- 帮我搜索 WPS
- 查一下浏览器应用
- 找一下可以做笔记的软件

### 2) 安装应用

- 帮我安装 WPS
- 安装 `cn.wps.wps-office`
- 我要安装这个应用的最新版本

### 3) 卸载应用

- 卸载 `cn.wps.wps-office`
- 帮我删除这个应用

### 4) 检查更新

- 检查一下有哪些应用可以更新
- 看看有没有可升级的软件

### 5) 执行升级

- 把可更新的应用全部升级
- 只升级 `org.deepin.calculator`

### 6) 查看详情

- 查看 `cn.wps.wps-office` 的详情
- 这个应用有什么版本

### 7) 提交反馈

- 反馈一个问题：商店首页加载很慢
- 帮我提交一个使用建议

## 推荐对话方式

为了更快完成操作，建议在对话里尽量说清楚：

- 应用名或 `appId`
- 是否指定版本
- 是否指定架构（不写就用默认）
