# 验收清单与自测流程

## 验收清单

- 在线搜索与应用列表始终使用 Web API。
- 可按关键词、分类、架构、语言、分页查询。
- 可查看应用详情（截图、标签、描述、版本、架构、运行时）。
- 安装支持指定版本、模块、仓库、架构与本地文件。
- 安装或卸载完成后调用 `/app/saveInstalledRecord`。
- 更新检查通过 `/app/appCheckUpdate` 并支持升级单个或全部。
- 反馈使用 `/web/suggest` 或 `/visit/suggest`。
- 安装失败输出 `ll-cli` 错误摘要并给出替代方案。

## 自测流程

1. 搜索应用：用关键词检索并展示候选与 `appId`。
2. 查看详情：确认详情接口返回字段齐全。
3. 安装应用：执行 `ll-cli install` 并完成上报。
4. 卸载应用：执行 `ll-cli uninstall` 并完成上报。
5. 检查更新：调用更新检查接口并执行升级。
6. 提交反馈：调用反馈接口并确认成功。

## 关键输出规范

- 列表输出必须含 `appId`。
- 多版本/多架构时必须要求用户确认。
- 失败时给出清晰可执行的下一步。
